# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added
- Initial release of PayHere PHP SDK
- Payment request builder with fluent interface
- Secure hash generation for payment requests
- Payment notification handler with verification
- Support for sandbox and live environments
- Comprehensive examples and documentation
- PSR-4 autoloading support
- MIT License

### Features
- Create payment requests
- Generate payment forms
- Auto-redirect to PayHere
- Verify payment notifications
- Handle all payment statuses (success, pending, failed, canceled, chargedback)
- Extract payment details from notifications
- Custom fields support
- Multiple currency support

### Documentation
- Complete README with API reference
- Quick start guide
- Working examples for checkout, notification handling, and return pages
- Inline code documentation
